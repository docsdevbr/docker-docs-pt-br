As versões antigas da referência do arquivo Compose foram movidas para a
[branch V1 do repositório do Compose](https://github.com/docker/compose/tree/v1/docs).
Elas não estão mais sendo mantidas ativamente.

A versão mais recente e recomendada do formato de arquivo Compose é definida
pela [Especificação do Compose](_index.md).
Este formato combina as versões 2.x e 3.x e é implementado a partir do **Compose
1.27.0**.
Para mais informações, consulte o
[Histórico e desenvolvimento do Docker Compose](/manuals/compose/intro/history.md).
